Undergraduate Research with CIIGAR Lab

Supervising Professor: Dr. David Roberts

Description

Training guide dogs requires an average of $50,000 invested per dog. With the failure rate around 60%, a method of maximizing the success of an individual dog and minimizing the time spent training unsuitable dogs would save thousands of dollars and increase the availability of successful guide dogs. The CIIGAR lab under the guidance of Dr. David Roberts and Dr. Alper Bozkurt has been compiling environmental data on hundreds of puppies through means of an electronically-fitted collar worn by the dogs when they are living with their volunteer raisers with goals of determining the ideal environment for raising guide dogs and connecting an individual dog’s responses to its likelihood of success in the program. My project is specifically aimed at developing location-aware analytics and a system of presentation that allows for user interaction.

Results

For the first project, I created a python and SQL script that cleaned the data in each table, interpolated missing values, and set the time stamps to a consistent time step over all the tables. After the tables were cleaned and standardized, I wrote another script to join them on tattoos and time stamps to create a large table to aid in lookups and comparing data from multiple sources to each other. Unfortunately, we ran into difficulties with resource consumption and were unable to drill down to the detail we were planning on. Dr. Roberts has plans to remedy these issues in the future.
For the second project, I was successful in clustering, graphing, and displaying the GPS data points.
The last component of my project is a summary of the results of the clustering and a tool to retrieve information from Google’s Maps API about specific coordinates. Each time the clustering algorithm is run on a tattoo, it generates a text file with the summary of the clusters, as well as provides the option for retrieving more information from Google using their Maps API. Future projects will include automating the process of gathering useful information from Google and automatically generating a map with the coordinates plotted on it.
These components together form the basis for a system where the GPS coordinates are grouped into clusters based on location and then connected with the other data points and information from Google to create a snapshot of the environment and the response of the dog, taking into account location and time.
